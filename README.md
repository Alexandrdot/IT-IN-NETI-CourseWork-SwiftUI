# IT-IN-NETI-CourseWork-SwiftUI
Объект исследования: процесс организации и проведения профориентационного мероприятия «IT в NETI» в Новосибирском Государственном Техническом Университете (НГТУ).
Предмет исследования: алгоритм и информационная система для автоматизированного распределения участников-школьников по мастер-классам на основе их приоритетов и ограничений мероприятий.
Цель курсовой работы: спроектировать и разработать модуль автоматизированного распределения участников для информационной системы профориентационного мероприятия «IT в NETI», повышающий эффективность организации и удовлетворенность участников.
